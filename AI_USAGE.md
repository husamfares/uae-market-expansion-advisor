# AI Tool Usage Disclosure

I used **ChatGPT** during development as an assistant for:

- architecture brainstorming and workflow planning
- code review and debugging support
- documentation refinement
- learning and applying the Agno framework, since I had not used it before

ChatGPT was especially useful in helping me map the overall system flow and think through the architecture step by step. However, the first suggestions were often basic, so I refined the design with my own updates and decisions. This included important additions such as the planner-based routing idea, the structured knowledge layer, the orchestrator flow, the deterministic validator, and many smaller implementation details across the project.

I personally implemented and fully understand the system architecture, agent structure, orchestration logic, tool usage, and evaluation process. I did not rely on blind copy-paste. Even when code was suggested, I preferred to write and adapt it myself so that I could fully understand how it works and be able to implement similar logic later on my own.

All final design decisions, architecture choices, and implementation structure were actively reviewed and understood by me as part of building the project.
